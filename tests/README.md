# dcrypt-tests

Integration test suite and verification harness for [dcrypt](https://github.com/ioi-foundation/dcrypt).

## Overview

This crate contains:
1.  **Integration Tests:** Verifying interoperability between crates.
2.  **ACVP Vectors:** Test harness for NIST Automated Cryptographic Validation Program vectors.
3.  **Constant-Time Verification:** Statistical tests to detect timing side-channels in core primitives.

This crate is for development and testing purposes only and is not intended for publication or direct use.
